# Content-Type Specific Adaptations

> Each content category has its own visual theme, colors, outfits, props, and doodles.

---

## 1. AI Automation Content (PRIMARY)

```
THEME COLORS:      Electric blue (#00D4FF), white, cyan, soft purple
CHARACTER OUTFIT:   Smart casual — polo shirt or button-down, modern look
KEY PROPS:          Laptop, robot icons, AI dashboard on screen, chatbot interface
INFO ELEMENTS:      Automation stats, time-saved numbers, workflow diagrams (simplified)
BACKGROUND:         Tech office, co-working space, modern desk setup
SIGNATURE DOODLES:  Robot emoji 🤖, brain with circuits, gear icons, lightning bolt
TYPICAL SCENES:     Demo of auto-replies, before/after time usage, lead capture flow
```

---

## 2. Web Design Content (SECONDARY)

```
THEME COLORS:      Purple (#7B2FFF), cyan, gradient blues, white
CHARACTER OUTFIT:   Creative casual — hoodie or designer jacket
KEY PROPS:          Design mockups on screen, color palettes, browser windows
INFO ELEMENTS:      Before/after website comparisons, mobile-responsive mockups
BACKGROUND:         Design studio, creative workspace, dual-monitor setup
SIGNATURE DOODLES:  Cursor arrow, browser icon, color wheel, paintbrush
TYPICAL SCENES:     Ugly website → beautiful redesign, mobile responsiveness demo
```

---

## 3. Graphics & Branding Content (TERTIARY)

```
THEME COLORS:      Yellow (#FFD700), orange (#FF9800), teal, vibrant mix
CHARACTER OUTFIT:   Creative — graphic tee, artistic accessories
KEY PROPS:          Logo designs, brand identity boards, color swatches, print materials
INFO ELEMENTS:      Before/after brand transformations, design principles
BACKGROUND:         Creative studio, colorful walls, design mood board visible
SIGNATURE DOODLES:  Paintbrush, color drops, crown (premium), sparkle stars
TYPICAL SCENES:     Logo redesign reveal, brand identity system showcase
```

---

## 4. Motivational / Personal Brand Content

```
THEME COLORS:      Gold (#FFD700), navy, white, warm amber
CHARACTER OUTFIT:   Sharp — blazer or kurta (depending on theme)
KEY PROPS:          Achievement symbols, journey milestones, growth metaphors
INFO ELEMENTS:      Inspirational quotes, journey timeline, milestone markers
BACKGROUND:         Sunrise/golden hour setting, or clean elevated space
SIGNATURE DOODLES:  Mountain peak, flag, trophy, fire emoji 🔥, crown
TYPICAL SCENES:     Personal journey frames, achievement celebration, advice sharing
```

---

## 5. Myth-busting / Trending Content

```
THEME COLORS:      Red (for myths), green (for facts), white
CHARACTER OUTFIT:   Casual relatable — t-shirt and jeans
KEY PROPS:          "MYTH vs FACT" comparison boards, magnifying glass
INFO ELEMENTS:      ❌ Myth cards (red) vs ✅ Fact cards (green)
BACKGROUND:         Bright room with debate-stage feel or classroom vibe
SIGNATURE DOODLES:  Cross marks, check marks, magnifying glass, detective hat
TYPICAL SCENES:     Myth reveal with red X, fact confirmation with green check
```

---

## 6. Client Testimonial / Case Study Content

```
THEME COLORS:      Green (#00FF88), blue, gold, white
CHARACTER OUTFIT:   Professional — button-down or semi-formal
KEY PROPS:          Client results dashboard, growth graph, testimonial quote card
INFO ELEMENTS:      Before/after stats, client quote, revenue/growth numbers
BACKGROUND:         Clean office, meeting room, professional setting
SIGNATURE DOODLES:  Upward arrow, star reviews, handshake, trophy
TYPICAL SCENES:     Client problem → solution → result transformation
```

---

## 7. Festival / Cultural Content (Seasonal)

```
THEME COLORS:      Traditional vibrant — saffron, green, maroon, gold
CHARACTER OUTFIT:   Ethnic wear — kurta, festive clothing
KEY PROPS:          Festival decorations (diyas, rangoli, lights, sweets)
INFO ELEMENTS:      Festival wishes, special offer announcement
BACKGROUND:         Decorated space matching festival theme
SIGNATURE DOODLES:  Diya, fireworks, festive patterns, stars
TYPICAL SCENES:     Festival greeting with brand touch, seasonal offer reveal
```
