# Visual Elements — Detailed Style Guide

> Complete specifications for every visual element used in image frames.

---

## Speech Bubbles

```
WHEN TO USE:   When character is "speaking" a key point or punchline
SHAPE:         Cloud-shaped with tail pointing to character's mouth
BACKGROUND:    White or cream with rounded corners
BORDER:        Subtle dark outline (1-2px)
FONT:          Bold, hand-written style (like comic lettering)
TEXT RULES:    
  - Max 2-3 lines per bubble
  - Hinglish (Hindi + English mix)
  - Key words in RED or BLUE for emphasis
  - Emojis allowed at end of sentence
SIZE:          Proportional — not larger than character's head
POSITION:      Above or beside character, tail pointing to mouth

EXAMPLES:
  - "Bhai automation lagao, 20 ghante bach jayenge! 🤖"
  - "Website se trust banao, leads apne aap aayenge! 🚀"
  - "Ye galti mat karna! ❌"
```

## Whiteboards & Flipcharts

```
WHEN TO USE:   For listing tips, steps, statistics, comparisons
TYPE:          Standing flipchart on easel, or wall-mounted whiteboard
SURFACE:       White/cream board
TEXT STYLE:    Hand-written with markers (blue/black primary, red/green accents)
CONTENT:       
  - Title underlined at top
  - Bullet points or numbered list (3-5 items max)
  - Color coding: Green ✅ for benefits, Red ❌ for problems
  - Small icons/emojis beside each point
CHARACTER:     Standing beside it, pointing or writing on it
SIZE:          30-40% of frame width, beside the character

EXAMPLE:
  Title: "AI Automation Ke 3 Fayde"
  ✅ 20+ Hours/Week Saved
  ✅ 0 Missed Leads  
  ✅ 24/7 Auto Replies
```

## Floating Info Cards / Badges

```
WHEN TO USE:   Surrounding character to show multiple related points
SHAPE:         Rounded rectangles with soft 3D shadow/elevation
SIZE:          Small cards (15-20% of frame width each)
LAYOUT:        Float around character in arc, scatter, or semi-circle
CONTENT:       Icon + Short text (3-5 words max) per card
COLORS:        Different pastel/vibrant color per card (no two same)
CONNECTOR:     Small arrows or dotted lines from character to cards
COUNT:         4-6 cards maximum per frame

EXAMPLE CARDS:
  🎯 "Target Audience"     (blue card)
  📊 "Data Driven"         (green card)
  🚀 "Fast Growth"         (orange card)
  💡 "Smart Solutions"     (purple card)
  ⚡ "Quick Response"      (yellow card)
```

## Sticky Notes

```
WHEN TO USE:   Fun facts, side comments, humor, casual info
SHAPE:         Small tilted squares with curled corner effect
COLORS:        Pastel yellow, pink, blue, green
TEXT:           Handwritten-style, short (1-2 lines)
PLACEMENT:     On desks, monitors, walls — peripheral positions
SIZE:          Small (8-12% of frame)

EXAMPLES:
  Yellow note: "Coffee + Code = Magic ☕"
  Pink note: "Client happy hai! 🎉"
  Blue note: "Deadline: Yesterday 😅"
```

## Device Screens

```
WHEN TO USE:   Showing results, dashboards, websites, before-after
DEVICES:       Laptop, phone, tablet, desktop monitor
SCREEN RULES:
  - Content MUST be clearly visible and readable
  - Screen should be BRIGHT (not dark mode)
  - Content matches the scene's topic
  - Can show: growth graphs, chat interfaces, website mockups,
    notification panels, analytics dashboards
ANGLE:         Slight perspective tilt (natural desk view) or straight-on
SIZE:          20-35% of frame, positioned near character
```

## Decorative Doodles & Sketches

```
COUNT:         3-5 per frame (MANDATORY)
STYLE:         Hand-drawn, sketch-like, organic placement
COLOR:         White (on colored backgrounds) or accent color
SIZE:          Small relative to character (5-10% of frame each)
PLACEMENT:     Scattered naturally, NOT in grid or uniform pattern

AVAILABLE DOODLES:
├── ⭐ Stars and sparkles      → around success/highlight elements
├── 💡 Lightbulbs              → ideas and tips
├── 🚀 Rockets                 → growth and speed
├── ↗️ Arrows (up/diagonal)    → pointing to key info
├── ❗ Exclamation marks        → emphasis and surprise
├── ❓ Question marks           → curiosity and problems
├── 🎯 Targets/bullseyes       → goals and precision
├── ✨ Glitter/shine effects    → achievements and polish
├── 📊 Mini charts/graphs       → data visualization
├── ❌ Cross marks              → don'ts and problems
├── ✅ Check marks              → do's and solutions
├── 💬 Small thought clouds     → secondary thoughts
├── ⚡ Lightning bolts          → speed and energy
├── 🔥 Fire                     → trending and hot topics
├── 👆 Pointing fingers         → directing attention
├── 💪 Flexed arms              → strength and confidence
├── 🎉 Confetti/party           → celebration
├── ⏰ Clock/timer              → time and urgency
├── 💰 Money/coins              → savings and revenue
└── 🔔 Bell/notification        → alerts and attention

SELECTION GUIDE:
  Problem scenes → ❗❓❌⚡⏰
  Solution scenes → ✅💡🚀🎯✨
  Stats scenes → 📊✅⭐💰↗️
  CTA scenes → 👆🚀✨🔔💬
  Hook scenes → ❗❓🔥⚡⭐
```

## Branding Elements

```
INSTAGRAM HANDLE: "@ravi.digital.solutions"
  Position:   Bottom-left or top-right (consistent within a video)
  Font:       Clean, modern sans-serif
  Opacity:    70-80% (visible but not dominant)
  Color:      White or matching accent color
  Size:       Small but readable

CHARACTER PLACEMENT:
  Position:   Center or rule-of-thirds (left or right third)
  Size:       Minimum 40-60% of frame height
  Focus:      Character is ALWAYS the primary focal point
  Face:       Clearly visible, not obscured by text/elements

BRAND ACCENT COLORS (use at least ONE per frame):
  Electric Blue  #00D4FF  → Tech, solutions, AI
  Purple         #7B2FFF  → Creative, design, premium
  Green          #00FF88  → Success, results, stats, growth
  Orange         #FF6B35  → Problems, warnings (ONLY in Part A scenes)
  Yellow/Gold    #FFD700  → Highlights, attention, achievements
  
NOTE: These are ACCENT colors. Background & overall tone MUST stay LIGHT.
```
