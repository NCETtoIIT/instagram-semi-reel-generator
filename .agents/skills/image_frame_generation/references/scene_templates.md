# Scene-by-Scene Prompt Templates

> Complete templates for each of the 6 standard reel scenes.

---

## SCENE 1: HOOK (Part A)
**Purpose:** Grab attention, introduce the problem/topic  
**Recommended Style:** Style-C (Caricature) or Style-B (Photo + Overlay)

```
PROMPT STRUCTURE:
"[STYLE-C/B], vertical 9:16 (1080x1920px).

CHARACTER: 3D caricature of young Indian male (short dark hair, light 
beard/goatee), wearing [TOPIC-APPROPRIATE casual outfit], with a 
[SURPRISED/CURIOUS] expression, [POSE: scratching head / looking shocked / 
pointing at something off-screen].

SCENE: Bright [RELEVANT SETTING] with natural daylight. [1-2 key props 
related to the topic].

INFORMATION: Large bold text overlay at top: '[HOOK HEADLINE IN HINGLISH]'. 
[Optional: floating emoji or icon related to the topic].

DECORATIVE: Exclamation marks, question marks, and attention arrows drawn 
in sketch style around the character.

MOOD: Attention-grabbing, curious, energetic.
COLOR PALETTE: [3-4 bright colors matching topic]."
```

---

## SCENE 2: PAIN/PROBLEM (Part A)
**Purpose:** Deepen the problem, make audience feel the pain  
**Recommended Style:** Style-B (Real photo + cartoon showing frustration)

```
PROMPT STRUCTURE:
"[STYLE-B], vertical 9:16 (1080x1920px).

CHARACTER: 2D cartoon of young Indian male (short dark hair, goatee) placed 
in a real workplace photo. Wearing [OUTFIT], with a [STRESSED/FRUSTRATED] 
expression. [POSE: facepalm / slumped at desk / pulling hair / staring at 
overwhelming screen].

SCENE: Bright but slightly messy real workspace — papers on desk, multiple 
tabs open on screen, phone buzzing. The environment should still be LIGHT 
(not dark), but feel chaotic.

INFORMATION: Speech bubble from character saying: '[PAIN POINT IN HINGLISH]'. 
[Optional: red X marks or warning elements around the mess].

DECORATIVE: Swirl marks around head (confusion), small lightning bolts 
(stress), red ❌ marks near the problems.

MOOD: Relatable frustration, 'I've been there' feeling.
COLOR PALETTE: Warm tones, soft orange/red accents (problem colors), 
bright background."
```

---

## SCENE 3: TRANSITION (Part A → B)
**Purpose:** Bridge from problem to solution, show the contrast  
**Recommended Style:** Style-C (Caricature with split/comparison)

```
PROMPT STRUCTURE:
"[STYLE-C], vertical 9:16 (1080x1920px).

CHARACTER: 3D caricature of young Indian male (same features), now with a 
[KNOWING SMILE / CONFIDENT WINK] expression, wearing [SLIGHTLY UPGRADED 
OUTFIT from Scene 2]. Pose: pointing from one side to another, or holding 
up a solution (laptop/tool/idea bulb).

SCENE: Split-feel composition — left side slightly chaotic (faded/desaturated), 
right side clean and bright (vivid colors). Or: character standing at a 
crossroads/decision point.

INFORMATION: Floating comparison cards:
  LEFT: '❌ Manual Way' (red tinted)
  RIGHT: '✅ Smart Way' (green/blue tinted)
Optional: Arrow showing transformation from left → right.

DECORATIVE: Lightbulb above character's head (idea moment), stars on the 
'smart' side, dimmed elements on the 'manual' side.

MOOD: Revelation, turning point, 'aha moment'.
COLOR PALETTE: Transition from warm orange → cool blue/green."
```

---

## SCENE 4: SOLUTION (Part B)
**Purpose:** Present the solution clearly  
**Recommended Style:** Style-C (Caricature with educational elements)

```
PROMPT STRUCTURE:
"[STYLE-C], vertical 9:16 (1080x1920px).

CHARACTER: 3D caricature of young Indian male (same features), confident 
smile, wearing [PROFESSIONAL/SMART OUTFIT]. Pose: presenting/pointing at 
a whiteboard, or giving thumbs up beside a screen showing the solution.

SCENE: Bright, clean, modern workspace. Whiteboard or digital screen showing 
the solution clearly. Organized desk, good lighting, professional feel.

INFORMATION: Whiteboard/screen showing:
  '[SOLUTION TITLE]'
  '✅ Point 1: [KEY BENEFIT IN HINGLISH]'
  '✅ Point 2: [KEY BENEFIT IN HINGLISH]'
  '✅ Point 3: [KEY BENEFIT IN HINGLISH]'
Speech bubble: '[SOLUTION ONE-LINER IN HINGLISH]'

DECORATIVE: Green checkmarks, star sparkles around solution elements, 
small rocket icon (growth), thumbs up doodle.

MOOD: Confident, clear, professional, helpful.
COLOR PALETTE: Electric blue (#00D4FF), white, green accents (#00FF88)."
```

---

## SCENE 5: PROOF/STATS (Part B)
**Purpose:** Show results, statistics, social proof  
**Recommended Style:** Style-C (Caricature with data visualization)

```
PROMPT STRUCTURE:
"[STYLE-C], vertical 9:16 (1080x1920px).

CHARACTER: 3D caricature of young Indian male, excited/proud expression, 
wearing [OUTFIT]. Pose: celebrating or pointing at impressive numbers/graphs.

SCENE: Clean, minimal background (soft gradient or blurred bright office). 
Focus should be on the DATA being presented.

INFORMATION: Large, bold stats prominently displayed:
  '[BIG NUMBER/STAT 1]' (e.g., '20+ Hours/Week Saved')
  '[BIG NUMBER/STAT 2]' (e.g., '0 Missed Leads')
  '[BIG NUMBER/STAT 3]' (e.g., '3x Faster Response')
Each stat in a colorful card/badge with icon.
Optional: Growth chart arrow going up.

DECORATIVE: Stars and sparkles around the stats, upward trending arrows, 
trophy/medal icons, celebration confetti.

MOOD: Impressive, proof-driven, success celebration.
COLOR PALETTE: Green (#00FF88) for stats, gold (#FFD700), white, 
bright blue."
```

## SCENE 6: CTA (Part B)
**Purpose:** Call to action — tell audience what to do next  
**Recommended Style:** Style-C (Caricature, face-focused for brand recall)

```
PROMPT STRUCTURE:
"[STYLE-C], vertical 9:16 (1080x1920px).

CHARACTER: 3D caricature of young Indian male, warm friendly smile, waving 
or pointing at the viewer. Wearing [CLEAN, MEMORABLE OUTFIT]. This should be 
the MOST RECOGNIZABLE version of the character.

SCENE: Clean, bright gradient background (soft blue → white, or pastel blend). 
Minimal distractions — focus on character and CTA.

INFORMATION: 
  Main CTA text: 'DM karo \"[KEYWORD]\" 📩' (large, bold)
  Handle: '@ravi.digital.solutions'
  Subtext: '[WHAT THEY'LL GET — e.g., Free Automation Audit]'
  
BRANDING: Handle prominently displayed. Character's face is the central 
focus — maximum brand recognition.

DECORATIVE: Pointing hand doodle toward CTA, sparkles, DM envelope icon, 
small rocket.

MOOD: Inviting, friendly, clear, action-oriented.
COLOR PALETTE: Brand blue (#00D4FF), white, soft purple (#7B2FFF), 
green accent."
```

---

## Scene Flow Summary

```
Scene 1 (HOOK)       → Grab attention → Style B/C → Surprised/Curious
Scene 2 (PAIN)       → Feel the problem → Style B → Frustrated/Stressed  
Scene 3 (TRANSITION) → Aha moment → Style C → Knowing smile
Scene 4 (SOLUTION)   → Present answer → Style C → Confident
Scene 5 (STATS)      → Show proof → Style C → Proud/Excited
Scene 6 (CTA)        → Take action → Style C → Friendly/Inviting
```
