## 📌 SECTION 1: VISUAL STYLE DEFINITION

### 1.1 Style Formula

```
FINAL STYLE = 98% HYBRID-D + 2% STYLE-A

Where:
├── HYBRID-D = Mix of:
│   ├── STYLE-B: Real Photo + Cartoon Character Overlay
│   │   → Real workplace/environment photo background
│   │   → 2D cartoon character placed naturally in the scene
│   │   → Comic-style speech bubbles, doodle elements
│   │   → Natural lighting, bright & colorful
│   │
│   └── STYLE-C: 3D Caricature (Main Style)
│       → 3D rendered character with enlarged head
│       → Real facial features preserved, cartoon body proportions
│       → Bright isometric/3D backgrounds
│       → Hand-drawn doodle overlays
│
└── STYLE-A: Flat Vector (2% for text-heavy info screens only)
    → Bold flat typography, clean layout
    → Only when pure data/stats need display
```

### 1.2 Character Design (CONSISTENT across all scenes)

```
CHARACTER:
├── Face: Young Indian male, early 20s
├── Hair: Short, dark, neatly styled
├── Facial Hair: Light beard/goatee (signature feature)
├── Build: Medium-slim
├── Expression: CHANGES per scene (shocked → stressed → knowing → confident → excited → friendly)
├── Outfit: Dynamic per topic (see outfit matrix below)
└── CRITICAL: Same face features in EVERY scene
```

### 1.3 Color System

```
BACKGROUND:     Light & Bright (pastel, natural daylight, white/cream)
                NEVER dark/black/noir backgrounds!

ACCENT COLORS (used sparingly):
├── Electric Blue  #00D4FF → Tech, Solutions, CTAs
├── Purple         #7B2FFF → Creative, Design, Gradients
├── Green          #00FF88 → Success, Stats, Results
├── Orange         #FF6B35 → Problems ONLY (frustration scenes)
├── Yellow/Gold    #FFD700 → Highlights, Celebrations
└── Overall tone:  LIGHT, BRIGHT, VIBRANT, ENERGETIC
```

### 1.4 Mandatory Elements in Every Frame

```
EVERY FRAME MUST INCLUDE:
├── 3-5 hand-drawn doodle elements (sparkles, arrows, !, lightbulbs)
├── At least 1 information delivery element (speech bubble, whiteboard, cards)
├── Instagram handle '@ravi.digital.solutions' (semi-transparent)
├── Vertical 9:16 format (1080×1920px)
└── NO stock photos, NO clipart, NO dark backgrounds
```
